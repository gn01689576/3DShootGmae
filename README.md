# 3D射擊小遊戲
學習途中一邊做出來的遊戲，有一點大雜燴的感覺
這邊放上程式碼跟APK

## 玩法
1. 左邊搖桿移動，大顆按鍵發射子彈，小顆按鍵周圍發射一圈子彈。
2. 藍色怪物碰到會爆炸，觸手怪物遠程攻擊，寶相怪物近距離攻擊。
